PAGINA WEB PROGRESIVA

    PWAs se basan en conceptos sencillos:

  • Diseño Responsive

    • Animaciones CSS y frameworks específicos para dar la apariencia de app móvil nativa
  • Service Workers:

    • secuencia de comandos que tu navegador ejecuta en segundo plano, separado de una página web, abriéndoles la puerta a funciones que no necesitan una página web ni interacción de usuario.
  • App Shell

    • s un modelo que separa la funcionalidad del contenido para cargarlos por separado. De esta manera, podemos tener la aplicación cacheada para usarla off-line y que cargue muy rápido, y por otro lado el contenido que puede estar también precargado y actualizarse cuando haya conexión.
  • Manifiesto de Aplicación

    • es un archivo JSON simple que permite que el desarrollador , pueda controlar cómo se muestra la app al usuario en áreas donde normalmente ven apps nativas guardando un marcador de sitio, en la pantalla de inicio de un dispositivo.
