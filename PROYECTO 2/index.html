¿Qué es una Aplicación Web Progresiva o PWA?


las aplicaciones web progresivas están a medio camino entre de dos anteriores que son ,web y nativa. son básicamente páginas web, pero mediante el uso de Service Workers y otras tecnologías se comportan más como aplicaciones normales que como aplicaciones web. Este tipo de aplicaciones basadas en la web, hacen uso de las últimas tecnologías disponibles en los navegadores, para ofrecer una experiencia lo más completa posible, con una interfaz y una cantidad de funcionalidades que muy poco tienen que envidiar a las de las apps nativas. Esto, se suma a un rendimiento que suele estar bastante por encima gracias a que son capaces de consumir una menor cantidad de recursos, y en muchas ocasiones, existe la posibilidad de trabajar con ellas sin necesidad de conexión a Internet. También ocupan menos espacio, así que utilizar este tipo de apps es una de las mejores formas de liberar espacio en Android.
